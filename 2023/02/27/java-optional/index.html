Optional 详解

Optional

我们都知道在 Java8 中引入了很多有用的 API, 其中就包含了optional。只要稍加运用就能够发现, 这个类可以大大减少开发者的代码输出, 也可以使代码更加的优雅, 让代码阅读者看见就想亲你两口

1. 能做什么?

在我们实际业务开发中, 大部分情况下交互数据库或者调用别人的接口时, 都不能保证对方传递过来的参数是否为空, 从而导致每次都要判断空指针。
